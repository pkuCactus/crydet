# 数据增强策略与流程

本文档详细描述婴儿哭声检测系统的数据增强策略，包括标签感知的Mixup增强、Sox音频效果链、噪声注入和动态增益调整。

## 整体流程图

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         AudioAugmenter.augment()                            │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ Step 1: 标签判断与反转                                                        │
│ ┌─────────────────────────────────────────────────────────────────────────┐ │
│ │ 输入: waveform y, label                                                 │ │
│ │                                                                         │ │
│ │  is_cry = (label == 'cry')                                              │ │
│ │  if not is_cry and random() < 0.5:                                      │ │
│ │      y = reverse(y)  // 非哭声50%概率时间反转                             │ │
│ └─────────────────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ Step 2: Mixup前置增强 (70%概率)                                              │
│ ┌─────────────────────────────────────────────────────────────────────────┐ │
│ │  if random() < 0.7:                                                     │ │
│ │      y = do_mixup(y, label)                                             │ │
│ └─────────────────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ Step 3: Sox效果链增强                                                        │
│ ┌─────────────────────────────────────────────────────────────────────────┐ │
│ │  判断是否增强:                                                             │ │
│ │    - Cry样本: 90%概率增强                                                 │ │
│ │    - 非Cry样本: 60%概率增强                                               │ │
│ │                                                                         │ │
│ │  效果链 (随机顺序):                                                        │ │
│ │    ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐             │ │
│ │    │  Pitch  │───▶│  Reverb │───▶│ Phaser  │───▶│  Echo   │             │ │
│ │    │  (50%)  │    │  (80%)  │    │  (50%)  │    │ (非Cry) │             │ │
│ │    └─────────┘    └─────────┘    └─────────┘    └─────────┘             │ │
│ │                                                     (50%)                 │ │
│ │                                                                         │ │
│ │  + 噪声注入 (10%概率, SNR 10-30dB)                                        │ │
│ │  + 增益归一化 (90%概率)                                                   │ │
│ └─────────────────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ Step 4: Mixup后置增强 (30%概率)                                              │
│ ┌─────────────────────────────────────────────────────────────────────────┐ │
│ │  if not is_mixup_front:                                                 │ │
│ │      y = do_mixup(y, label)                                             │ │
│ └─────────────────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ Step 5: 动态增益调整                                                         │
│ ┌─────────────────────────────────────────────────────────────────────────┐ │
│ │  if is_aug and random() < 0.9:                                          │ │
│ │      if random() < 0.1:                                                 │ │
│ │          gain_db = uniform(0, 10)      // 10%概率小幅增益                 │ │
│ │      else:                                                              │ │
│ │          gain_db = gauss(0, 20)        // 90%概率高斯分布衰减              │ │
│ │          gain_db = clip(gain_db, -inf, -40)                             │ │
│ │      y = normalize(y) * gain_db                                         │ │
│ └─────────────────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
                              输出: 增强后波形
```

## Mixup增强详细流程

### Mixup规则矩阵

| 样本类型 | Mixup概率 | Mixup率分布 | 混合样本来源 | 能量约束 |
|---------|----------|------------|-------------|---------|
| **Cry** | 0.3 | 均值0.3, 标准差0.15, 裁剪至[0.1, 0.65] | 任意样本 | 混合样本能量 < 原始能量 - (3~10)dB |
| **非Cry** | 0.3 | 随机均匀分布 | 仅非Cry样本 | 无特殊约束 |

### Mixup流程图

```
┌─────────────────────────────────────────────────────────────────┐
│                      do_mixup(y, label)                         │
└─────────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│  1. 生成Mixup样本                                                │
│     ┌───────────────────────────────────────────────────────┐   │
│     │ y_mix = generate_mixup_sample(y, label)               │   │
│     │                                                       │   │
│     │  • 根据标签选择可用样本池                                │   │
│     │  • Cry: 所有标签均可                                    │   │
│     │  • 非Cry: 排除'cry'标签                                 │   │
│     │  • 从file_schedule_dict随机选择加载                      │   │
│     └───────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│  2. 能量约束 (仅Cry样本)                                         │
│     ┌───────────────────────────────────────────────────────┐   │
│     │ if is_cry:                                            │   │
│     │     original_db = get_db(y)                           │   │
│     │     mix_db = get_db(y_mix)                            │   │
│     │     if mix_db >= original_db:                         │   │
│     │         target_diff = uniform(3, 10)  // 目标差3-10dB  │   │
│     │         y_mix = y_mix * 10^(-(mix_db - original_db +   │   │
│     │                              target_diff) / 20)         │   │
│     └───────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│  3. 计算Mixup位置与混合                                          │
│     ┌───────────────────────────────────────────────────────┐   │
│     │ st = random_int(0, len(y) - len(y_mix))               │   │
│     │ y[st:st+len(y_mix)] += y_mix                          │   │
│     │ y = clip(y, -1, 1)                                    │   │
│     └───────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                            │
                            ▼
                     返回增强后波形
```

## Sox效果链配置

### 效果链结构

```
┌─────────────────────────────────────────────────────────────────────┐
│                     Sox Effect Chain                                │
│                     (随机顺序应用)                                    │
└─────────────────────────────────────────────────────────────────────┘

  ┌──────────────┐      ┌──────────────┐      ┌──────────────┐
  │    Pitch     │      │    Reverb    │      │   Phaser     │
  │    50%概率   │◄────►│    80%概率   │◄────►│    50%概率   │
  │  (-2~+2半音) │      │  随机混响参数  │      │  随机相位参数  │
  └──────────────┘      └──────────────┘      └──────────────┘
                                                       │
                                                       │ (仅非Cry)
                                                       ▼
                                               ┌──────────────┐
                                               │    Echo      │
                                               │    50%概率   │
                                               │  单回声效果    │
                                               └──────────────┘
```

### 各效果详细参数

#### Pitch (音高变换)
```python
pitch_rate = (random() - 0.5) * 4  # 范围: -2 到 +2 半音
```

#### Reverb (混响)
```python
{
    'reverberance': random() * 80 + 20,      # 20-100
    'high_freq_damping': random() * 100,      # 0-100
    'room_scale': random() * 100,             # 0-100
    'stereo_depth': random() * 100,           # 0-100
    'pre_delay': 0
}
```

#### Phaser (相位器)
```python
{
    'gain_in': random() * 0.5 + 0.5,          # 0.5-1.0
    'gain_out': random() * 0.5 + 0.5,         # 0.5-1.0
    'delay': random_int(1, 5),                # 1-5 ms
    'decay': random() * 0.4 + 0.1,            # 0.1-0.5
    'speed': random() * 1.9 + 0.1,            # 0.1-2.0 Hz
    'modulation_shape': random(['sinusoidal', 'triangular'])
}
```

#### Echo (回声 - 仅非Cry)
```python
{
    'gain_in': random() * 0.5 + 0.5,          # 0.5-1.0
    'gain_out': random() * 0.5 + 0.5,         # 0.5-1.0
    'n_echos': 1,
    'delays': [random_int(6, 60)],            # 6-60 ms
    'decays': [random() * 0.5]                # 0-0.5
}
```

## 配置参数汇总

### AugmentationConfig

| 参数 | 默认值 | 说明 |
|-----|-------|------|
| `cry_aug_prob` | 0.9 | Cry样本应用Sox效果链的概率 |
| `other_aug_prob` | 0.6 | 非Cry样本应用Sox效果链的概率 |
| `other_reverse_prob` | 0.5 | 非Cry样本时间反转概率 |
| `pitch_prob` | 0.5 | 音高变换效果概率 |
| `reverb_prob` | 0.8 | 混响效果概率 |
| `phaser_prob` | 0.5 | 相位器效果概率 |
| `echo_prob` | 0.5 | 回声效果概率（仅非Cry） |
| `noise_prob` | 0.1 | 噪声注入概率 |
| `gain_prob` | 0.9 | 动态增益调整概率 |

### MixupConfig

| 参数 | 默认值 | 说明 |
|-----|-------|------|
| `cry_mix_prob` | 0.3 | Cry样本Mixup概率 |
| `cry_mix_rate_mean` | 0.3 | Cry样本Mixup率均值 |
| `cry_mix_rate_std` | 0.15 | Cry样本Mixup率标准差 |
| `other_mix_prob` | 0.3 | 非Cry样本Mixup概率 |
| `mix_front_prob` | 0.7 | Mixup前置（效果链之前）概率 |

## 标签感知增强策略

### Cry样本增强策略

```
┌──────────────────────────────────────────────────────────────┐
│                     Cry 样本 (婴儿哭声)                        │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  Mixup增强 (30%概率)                                          │
│  ├── 混合率: 高斯分布 N(0.3, 0.15), 裁剪至 [0.1, 0.65]          │
│  └── 能量约束: 混合样本必须比原始低 3-10 dB                      │
│                                                              │
│  Sox效果链 (90%概率)                                          │
│  ├── 音高变换 (-2~+2半音)                                      │
│  ├── 混响 (高概率 80%)                                        │
│  ├── 相位器                                                   │
│  └── 回声 (不包含 - 保持哭声清晰度)                              │
│                                                              │
│  噪声注入 (10%概率, SNR 10-30dB)                               │
│                                                              │
│  动态增益 (90%概率)                                           │
│  └── 模拟不同距离/音量的哭声                                   │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### 非Cry样本增强策略

```
┌──────────────────────────────────────────────────────────────┐
│                   非Cry 样本 (环境/其他声音)                    │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  时间反转 (50%概率)                                           │
│  └── 增加时间不变性，防止模型过拟合时间模式                      │
│                                                              │
│  Mixup增强 (30%概率)                                          │
│  ├── 混合率: 均匀随机分布                                       │
│  └── 约束: 只能混合非Cry样本 (防止哭声污染)                      │
│                                                              │
│  Sox效果链 (60%概率)                                          │
│  ├── 音高变换                                                 │
│  ├── 混响                                                     │
│  ├── 相位器                                                   │
│  └── 回声 (包含 - 模拟环境反射)                                 │
│                                                              │
│  噪声注入 (10%概率)                                           │
│                                                              │
│  动态增益 (90%概率)                                           │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## 设计要点

1. **标签差异化处理**: Cry和非Cry采用不同的增强策略，Cry更注重保持声学特征完整性，非Cry更注重多样性

2. **能量感知Mixup**: Cry样本混合时强制要求混合样本能量更低，模拟背景噪声而非主导声音

3. **样本隔离**: 非Cry样本Mixup时排除Cry样本，防止环境声音被哭声污染

4. **效果链动态组合**: 效果应用顺序随机，增加组合多样性

5. **时间反转数据增强**: 仅应用于非Cry，增加时间不变性训练
